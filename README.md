# Ganesh Portfolio Website

This is a personal portfolio website for Ganesh, built with plain HTML and CSS. It presents an AI/ML-focused student profile, skills, recent projects, certificates, and contact section in a clean single-page layout.

## Features

- Responsive one-page portfolio layout
- Sections for home, about, skills, projects, certificates, and contact
- Smooth scrolling navigation
- Reusable button, card, and tag styles
- Profile image support with `passport.jpg`

## Project Structure

- `index.html` - page content and section structure
- `style.css` - visual design, layout, and responsive styles
- `passport.jpg` - profile image used in the site

## Run Locally

Because this is a static site, no build step is required.

1. Open `index.html` directly in a browser, or
2. Serve the folder with any static server

Example with VS Code Live Server or any local web server works fine.

- Update text content in `index.html`
- Replace `passport.jpg` with your own image
- Adjust colors in the `:root` variables inside `style.css`
- Replace social links with your actual profile URLs
